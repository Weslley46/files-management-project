🚀 Projeto Finalizado: File Manager CLI
Acabei de concluir um gerenciador de arquivos completo via linha de comando desenvolvido em Node.js.
O projeto permite ao usuário realizar todas as operações básicas de gerenciamento de arquivos diretamente pelo terminal, de forma simples, intuitiva e robusta:
✅ Criar pasta automaticamente
✅ Criar novos arquivos
✅ Listar todos os arquivos da pasta
✅ Ler conteúdo de arquivos
✅ Escrever/editar conteúdo
✅ Deletar arquivos
Tecnologias utilizadas:

Node.js (com ES Modules)
File System (fs)
readline-sync (interface interativa)
Promises + async/await
Tratamento de caminhos com path

Foi um excelente exercício para aprofundar meus conhecimentos em manipulação do sistema de arquivos, construção de CLIs e migração de CommonJS para ES Modules (que deu um trabalhinho no começo 😂).
Estou cada vez mais apaixonado por backend e ferramentas que facilitam o dia a dia do desenvolvedor.
